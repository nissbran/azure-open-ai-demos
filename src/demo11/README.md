# Demo11

Microsoft Agents AI framework basic chat demo. This demo implements a simple chat application using the [Microsoft Agents AI](https://github.com/microsoft/Agents-for-net) framework with Azure OpenAI.

## To run

Use the instructions in the [main readme](../README.md) to deploy the infrastructure and set up environment variables or user secrets. It is also possible to use a local json file for configuration.

Navigate to the agent subdirectory and run:
```bash
cd src/demo11/agent
dotnet run
```
